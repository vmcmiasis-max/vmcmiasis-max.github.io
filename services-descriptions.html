АРХІТЕКТУРНЕ ПРОЄКТУВАННЯ

1. Проєктна пропозиція

Що це? Попередній варіант проєкту будівлі або спорудження, який представляє концептуальний підхід до розв'язування архітектурних та планувально-композиційних задач.

Для чого потрібна? Визначення працездатності пропозиції, стилістичного напрямку, геометричної та функціональної опцій; отримання згоди замовника та інвесторів перед розробленням детального проєкту.

2. Містобудівний розрахунок

Що це? Документальний опис інвестиційних намірів забудовника, що містить графічні матеріали та техніко-економічні показники (поверховість, площа, інтенсивність) майбутнього об'єкта.

Для чого потрібна? Для отримання містобудівних умов і обмежень від органів влади.

3. Ескіз намірів забудови земельної ділянки

Що це? Графічне зображення розташування майбутнього об'єкта на земельній ділянці з урахуванням існуючої забудови та комунікацій; попередня схема організації території.

Для чого потрібна? На етапі планування — для оцінки потенціалу ділянки та вибору найоптимальнішого розташування будівлі.

4. Робочий проєкт

Що це? Комплект конструкторської документації, що містить детальні креслення всіх конструктивних елементів, систем та комунікацій, розраховані на безпосередній виконавців робіт.

Для чого потрібна? Допуск до будівництва; координація розташування інженерних мереж та конструкцій; розрахунок кошторисної вартості та термінів будівництва.

7. Обґрунтування визначення меж і площі земельної ділянки

Що це? Документ, що обґрунтовує законність та правильність визначення меж ділянки; містить історичні дані, кадастрові планів та правові підстави.

Для чого потрібна? При спорах про межи ділянок; для реєстрації в земельному кадастрі з чіткою правовою основою.

8. Паспорт опорядження фасадів

Що це? Документ, що описує матеріали, виконання та мета­риали для опорядження зовнішніх поверхонь будівлі, включаючи колірну схему та спеціальні рішення.

Для чого потрібна? Для узгодження з органами охорони культурної спадщини та муніципалітетом; для наведення єдиного архітектурного вигляду міської забудови.

9. Розрахунок ідеальної земельної частки

Що це? Визначення частки кожного власника у спільній земельній ділянці, що розраховується пропорційно до площі його приміщень або інших показників.

Для чого потрібна? При розбиранні однієї земельної ділянки між кількома власниками будівель; для уточнення правового статусу спільного майна.

ГЕОДЕЗИЧНІ ПОСЛУГИ

1. Винесення меж земельної ділянки в натуру на місцевість

Що це? Геодезична послуга з визначення та встановлення на місцевості точних координат меж земельної ділянки за допомогою спеціальних геодезичних інструментів.

Для чого потрібна? При формуванні нової земельної ділянки, приватизації, оренді; для визначення меж на місцевості.

2. Викопіювання з чергового плану міста 1:2000

Що це? Виготовлення копії плану із офіційного міського плану в масштабі 1:2000.

Для чого потрібна? Для отримання інформації про розташування земельної ділянки в місті; формування документації для органів влади.

3. Виготовлення топографо-геодезичних планів масштабів 1:500

Що це? Детальний топографічний план ділянки, на якому зображені рельєф місцевості, існуючі комунікації, забудова та інші об'єкти.

Для чого потрібна? Основа для проектування будівель та інженерних систем; обов'язкова для розроблення проєкту.

4. Виготовлення виконавчого (контрольного) топографо-геодезичного плану

Що це? Топографічний план, що відображає фактичні розміри та розташування будівлі та комунікацій після завершення будівництва (Ас-Білт документація).

Для чого потрібна? Для реєстрації завершеного об'єкту в Державному земельному кадастрі та Реєстрі нерухомості.

5. Кадастрова зйомка та виготовлення кадастрового плану

Що це? Комплексна геодезична послуга, що включає обстеження ділянки та створення нормалізованого кадастрового плану.

Для чого потрібна? Обов'язкова для реєстрації земельної ділянки в Державному земельному кадастрі; для отримання кадастрового номера.

1. Проекти землеустрою щодо відведення земельних ділянок

Опис: Це обов'язкова землевпорядна документація, що розробляється сертифікованим інженером землевпорядником для формування нової земельної ділянки (приватизація, оренда) або зміни цільового призначення земельної ділянки.

Коли розробляється?

  • При безкоштовній приватизації
  • При оформленні оренди
  • При зміні цільового призначення
  • При продажу ділянок державної/комунальної власності

Склад документації:

  • Пояснювальна записка
  • Кадастровий план (межи, площа, конфігурація)
  • Відомості про обмеження у використанні земель

2. Робочі проекти землеустрою

Опис: Це документація, що обґрунтовує заходи з використання та охорани земель, містить технічні розрахунки, креслення та кошторис для реалізації. Документація розробляється для рекультивації, захисту від ерозії, зняття родючого шару та покращення земель на основі рішень власників або органів влади.

Вартість: За індивідуальним розрахунком.

3. Технічна документація із землеустрою щодо встановлення (відновлення) меж земельної ділянки в натурі (на місцевості)

Опис: Це вид документації із землеустрою, що розробляється для визначення або відновлення фактичних меж земельної ділянки на місцевості та фіксації їх у Державному земельному кадастрі.

Коли потрібна документація?

  • Приватизація: якщо громадянин користується ділянкою та має на ній право власності на житловий будинок
  • Відновлення меж: якщо межові знаки були втрачені або виникла суперечка із сусідами
  • Присвоєння кадастрового номера: для ділянок, на які право власності виникло до 2004 року, але відомості про них відсутні в кадастрі
  • Оренда: при передачі ділянок державної чи комунальної власності в користування (у певних випадках)

4. Технічна документація із землеустрою щодо встановлення меж частини земельної ділянки, на яку поширюється право суборенди, сервітуту

Опис: Документація для визначення меж частини земельної ділянки, що передається в суборенду або обтяжена сервітутом (правом проходу, прокладання комунікацій тощо).

6. Технічна документація із землеустрою щодо інвентаризації земель

Опис: Документація, що містить описування та оцінку стану земельних ділянок, їх характеристик та природних властивостей. Розробляється для уточнення даних про земельні ділянки та актуалізації інформації в Державному земельному кадастрі.

Вартість: За індивідуальним розрахунком.

7. Проекти землеустрою щодо встановлення (зміни) меж адміністративно-територіальних одиниць

Опис: Це документація, що визначає територіальний обсяг повноважень органів місцевого самоврядування та межи населених пунктів або районів.

🔹 Підстави розроблення: Створення нової адміністративної одиниці (села, селища, міста) • Зміни існуючих меж (розширення/зменшення території) • Включення нових земель відповідно до генерального плану.

8. Проекти землеустрою щодо встановлення меж територій територіальних громад

Опис: Це ключовий інструмент просторового планування, який визначає зовнішню межу всієї територіальної громади, що дає їй змогу законно розпоряджатися землями за межами населених пунктів.

🔹 Навіщо це громаді: Чітке розмежування з територіями сусідніх громад • Закріплення права на державні землі • Можливість збору земельного податку та оренди • Інвестиційна привабливість.

9. Проекти землеустрою щодо організації і встановлення меж територій природно-заповідного фонду та іншого природоохоронного призначення, оздоровлювального, рекреаційного, історико-культурного, лісогосподарського призначення, земель водного фонду та водоохоронних зон, обмежень у використанні земель та їх режимоутворюючих об'єктів

Опис: Це комплексний вид проектів, що регулюють межи та особливий режим господарювання на територіях із високою цінністю для держави.

🔹 Основні типи: Природно-заповідний фонд (ПЗФ) • Історико-культурне призначення • Водний фонд та водоохоронні зони • Лісогосподарське призначення.

🔹 Мета: Захист культурного шару, визначення прибережних захисних смуг, відмежування лісових масивів від сільгоспугідь та забудови.

10. Формування обмінних файлів

Опис: Обмінний файл (XML-файл) — це електронний документ установленої структури, що містить повну інформацію про земельну ділянку у цифровому форматі.

🔹 Містить: Межи ділянки (координати) • Площу • Цільове призначення • Дані про власника • Угіддя • Обмеження • Необхідна для реєстрації в державних реєстрах.

ФАСАДИ ТА ЗОВНІШНІЙ ВИГЛЯД

1. Ескіз рекламної конструкції

Що це? Графічне оформлення рекламної вивіски, біл-борду або іншої конструкції з урахуванням композиції, розмірів та розташування на фасаді.

Для чого потрібна? Для отримання дозволу на розміщення реклами; для узгодження з архітектурним виглядом будівлі та міста.

МІСТОБУДІВНІ ПОСЛУГИ

1. Комплексний план просторового розвитку

Опис: Комплексний план просторового розвитку території територіальної громади — це основний містобудівний документ на місцевому рівні та документація із землеустрою, що визначає стратегію розвитку, функціональне призначення земель, інфраструктуру та екомережу всієї території громади. Це обов'язковий документ для управління земельними ресурсами, який замінює окремі генплани населених пунктів, план зонування та детальні плани територій.

Вартість: За індивідуальним розрахунком.

2. Генеральний план населеного пункту

Опис: Генеральний план населеного пункту — це містобудівна та землевпорядна документація на місцевому рівні, що визначає довгострокову стратегію планування, забудови та функціонального використання території населеного пункту. Він затверджується для населеного пункту з урахуванням державних, громадських та приватних інтересів, регулюючи розвиток інфраструктури, зонування та забудову.

Вартість: За індивідуальним розрахунком.

3. Детальний план території

Опис: Детальний план території — це ключова містобудівна документація, що визначає планувальну організацію, функціональне призначення, параметри забудови та благоустрій окремої ділянки (кварталу, мікрорайону). Він уточнює генеральний план, визначає обмеження у використанні земель, інженерні мережі та межи обмежень, необхідні для будівництва, реконструкції чи землевідведення.

Вартість: За індивідуальним розрахунком.

4. Проект розподілу території кварталу

Опис: Проект розподілу території кварталу - це містобудівна документація, що розробляється для розмежування земельних ділянок у межах мікрорайону (кварталу) або його частини. Цей документ є ключовим для визначення меж прибудинкових територій та можливості приватизації або оренди землі мешканцями та власниками нерухомості.

Вартість: За індивідуальним розрахунком.

5. Розроблення бази геопросторових даних містобудівної документації

Опис: Розроблення бази геопросторових даних (БГД) є обов'язковим етапом створення містобудівної документації в Україні згідно з Постановою КМУ № 926. Це цифровий набір даних, який забезпечує точне просторове відображення проєктних рішень та їх інтеграцію до Державного містобудівного кадастру згідно з Наказом Міністерства розвитку громад та територій України № 56 від 22.02.2022 року "Про затвердження структури Бази геоданих містобудівної документації на місцевому рівні".

Вартість: За індивідуальним розрахунком.

6. Комплексний містобудівний аналіз

Опис: Комплексний містобудівний аналіз земельної ділянки — це всебічне дослідження території для визначення її потенціалу, обмежень та розробки рішень щодо забудови чи реорганізації. Він включає вивчення Комплексного плану просторового розвитку території, Генерального плану, для обґрунтування ефективного використання земельної ділянки.

Вартість: Аналіз однієї ділянки - 1780,00/2136,00 грн.
